Equivalent to open(2).

The full list of flags are:

  • EV_FS_O_APPEND
  • EV_FS_O_CREAT
  • EV_FS_O_DSYNC
  • EV_FS_O_EXCL
  • EV_FS_O_SYNC
  • EV_FS_O_TRUNC
  • EV_FS_O_RDONLY
  • EV_FS_O_WRONLY
  • EV_FS_O_RDWR

The full list of mode are:

  • EV_FS_S_IRUSR
  • EV_FS_S_IWUSR
  • EV_FS_S_IXUSR
  • EV_FS_S_IRWXU
Note
File always open in binary mode.
Parameters
[in]fileFile handle.

[in]pathFile path.
